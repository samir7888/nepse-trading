# 📁 Your Clean Investment System - File Guide

## 🎯 **ESSENTIAL FILES FOR YOUR 3000 NPR INVESTMENT**

### 📊 **MAIN INVESTMENT GUIDES** (Read These First!)

1. **`CORRECTED_RECOMMENDATIONS.md`** ⭐ **START HERE**

   - Your current stock recommendations
   - Exact prices and investment amounts
   - Fixed entry prices and budget management

2. **`YOUR_3000_NPR_INVESTMENT_STRATEGY.md`** ⭐ **COMPLETE GUIDE**

   - Your personalized investment plan
   - Step-by-step action plan
   - Risk management rules

3. **`BEGINNER_INVESTMENT_GUIDE.md`** ⭐ **LEARNING RESOURCE**

   - Complete beginner's guide to investing
   - How the system works
   - Do's and don'ts for new investors

4. **`STOCK_SELECTION_CRITERIA_EXPLAINED.md`** 📚 **UNDERSTANDING**
   - Why specific stocks are recommended
   - Technical analysis explanation
   - Selection criteria details

---

### 🖥️ **SYSTEM FILES** (The Engine Behind Recommendations)

5. **`beginner_trading_system.py`** 🔧 **MAIN PROGRAM**

   - Run this to get new recommendations
   - Generates Excel reports
   - Uses current market prices

6. **`config.py`** ⚙️ **SETTINGS**

   - Your 3000 NPR budget setting
   - Risk management parameters
   - Trading rules configuration

7. **`data_fetcher.py`** 📡 **DATA SOURCE**

   - Gets current stock prices
   - Fetches company information
   - Connects to NEPSE API

8. **`signal_generator.py`** 🎯 **ANALYSIS ENGINE**

   - Generates buy/sell signals
   - Calculates risk-reward ratios
   - Applies trading rules

9. **`backtesting_engine.py`** 📈 **PERFORMANCE TESTING**
   - Tests strategy on historical data
   - Calculates expected returns
   - Risk assessment

---

### 🚀 **EXECUTION FILES** (How to Run the System)

10. **`setup_and_run.bat`** 🏃 **EASY START**

    - Double-click to run everything
    - Installs requirements automatically
    - Runs the investment system

11. **`run_system.py`** 🎮 **MANUAL RUN**
    - Alternative way to run the system
    - More control over execution
    - Shows detailed progress

---

### 📁 **FOLDERS**

12. **`backend/`** 🏗️ **LOCAL SERVER** (Optional)

    - Local NEPSE API server
    - Provides faster data access
    - Not required for basic usage

13. **`output/`** 📊 **RESULTS**
    - Contains Excel reports
    - `beginner_trading_strategy.xlsx` - Your detailed analysis
    - Generated automatically when you run the system

---

### 📋 **SUPPORT FILES**

14. **`requirements.txt`** 📦 **DEPENDENCIES**

    - List of required Python packages
    - Used by setup script
    - Don't modify this

15. **`README.md`** 📖 **PROJECT INFO**

    - General project information
    - Installation instructions
    - Basic usage guide

16. **`start_backend.py`** 🔌 **SERVER STARTER** (Optional)
    - Starts local backend server
    - Only needed for advanced usage
    - Not required for beginners
